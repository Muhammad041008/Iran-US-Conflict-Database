# 🗂️ Iran–US Conflict Event Research Database (2018–2025)

## 📌 Project Overview
A structured research database of 96+ verified Iran-US conflict events 
from 2018 to 2025. Data collected from CNN, Reuters, Al Jazeera, IAEA, 
Pentagon, and PBS — cleaned, organized, and visualized in Excel and Power BI.

## 📁 Files in this Repository
| File | Description |
|------|-------------|
| `Iran_US_Conflict_Event_Database_MuhammadImran.xlsx` | Main Excel database — 96 events, 3 sheets |
| `IranUS_Events_PowerBI.csv` | Main CSV for Power BI import |
| `IranUS_YearSummary_PowerBI.csv` | Year-wise summary table |
| `IranUS_CategorySummary_PowerBI.csv` | Category breakdown |
| `IranUS_AggressorSummary_PowerBI.csv` | Aggressor analysis |
| `PowerBI_Dashboard_Guide_MuhammadImran.docx` | Step-by-step Power BI guide |

## 🛠️ Tools & Skills Used
- **Microsoft Excel** — Pivot Tables, VLOOKUP, INDEX-MATCH, COUNTIF, Conditional Formatting, VBA
- **Power BI** — KPI Cards, Bar Charts, Donut Charts, Slicers
- **Data Research** — CNN, Reuters, Al Jazeera, IAEA, Pentagon, PBS
- **Data Cleaning** — Duplicate removal, date standardization, source validation

## 📊 Excel Sheets
1. **Event Database** — 96 events with 13 columns (Date, Category, Severity, Source, etc.)
2. **Summary Analysis** — Year-wise, Severity, Category breakdowns (COUNTIF formulas)
3. **Data Cleaning Log** — 10 validation steps documented

## 🔍 Key Insights
- **2024–2025** saw the highest event frequency — Iran's first direct attack on Israel
- **Critical severity** events peaked during the April 2024 Iran-Israel exchange
- **Iran Proxy** groups responsible for 40%+ of total incidents
